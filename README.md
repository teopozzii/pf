# Personal finance dashboard

Dashboard app to analyze own income and consumption patterns, with more features than what is typically offered by bank apps. Reduce bias on spending habits, develop stronger financial responsibility.

Stable releaseas are downloadble from the Releases page of the repository as executables, both for Windows and MacOS.

MacOS users may need to manually entrust the app in the security settings:
- Enable execution from the terminal by running `chmod +x /path/to/BankStatementApp-MacOS`
- If the Mac blocks execution, go to `System Settings/Privacy & Security/Security` and select "Open Anyway" where the flag of the execution block is displayed.
